REST API сервис для командного управления задачами.

### Возможности

- JWT-аутентификация
- ролевая модель доступа
- история изменений задач
- Redis-кэширование
- SQL-отчёты
- Swagger / OpenAPI
- Docker Compose
- Prometheus
- Rate Limiting
- Circuit Breaker

## 📌 Subscription Service

Backend-сервис для управления подписками и расчёта финансовых показателей.

### Возможности

- Clean Architecture
- PostgreSQL
- сложные SQL-запросы
- фильтрация и агрегирование данных
- расчёт стоимости за произвольный период
- unit-тестирование
- Docker
