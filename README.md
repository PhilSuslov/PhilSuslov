# ⭐ Избранные проекты

## 📌 Task Service

REST API сервис для командного управления задачами.

### Возможности

- JWT-аутентификация;
- ролевая модель доступа;
- Redis-кэширование;
- SQL-отчеты;
- история изменений задач;
- командная работа;
- Swagger / OpenAPI;
- Docker Compose;
- Prometheus;
- Rate Limiting;
- Circuit Breaker.

---

## 📌 Subscription Service

Backend-сервис для учета подписок и расчета финансовых показателей.

### Возможности

- Clean Architecture;
- сложные SQL-запросы;
- агрегация данных;
- фильтрация;
- расчет стоимости за произвольный период;
- unit-тестирование;
- Docker.
